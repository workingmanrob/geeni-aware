# geeni-aware

mtdblock dump collected using a simple cat command:
```
cat /dev/mtdblock0 > /mnt/mtdblock0.bin
cat /dev/mtdblock1 > /mnt/mtdblock1.bin
cat /dev/mtdblock2 > /mnt/mtdblock2.bin
cat /dev/mtdblock3 > /mnt/mtdblock3.bin
cat /dev/mtdblock4 > /mnt/mtdblock4.bin
cat /dev/mtdblock5 > /mnt/mtdblock5.bin
cat /dev/mtdblock6 > /mnt/mtdblock6.bin
```
